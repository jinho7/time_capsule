# Time Capsule: 이거 기억나?

## 프로젝트 소개

**타임 캡슐: 이거 기억나?** 는 아날로그 감성과 추억 공유를 중심으로 한 플랫폼입니다. 사용자는 사진, 글, 영상을 기록하여 같은 세대의 사람들과 공유하고, 시간이 지나도 추억을 간직할 수 있습니다. 타임 캡슐 기능을 통해 일정 기간 후에 공개되는 콘텐츠를 설정할 수도 있습니다.

이 프로젝트는 단순한 SNS가 아니라, 세대 간 공감과 감성 공유를 목표로 합니다. 사용자들은 자신의 어린 시절, 학교생활, 가족 여행 등의 기억을 기록하고, 비슷한 경험을 가진 사람들과 소통할 수 있습니다.
